Todo List Maker

With this app you can make a simple Todo List. It has basic functionalities like adding a new task, mark as complete or remove the task.

This was a study assignment in which the goal was to interact with an API (json-server) to get/post data (which is then shown on the screen). It is built with HTML, CSS and JavaScript.

Note: The json-server is not running in the live version, so changes are lost on page refresh.

Programming languages used:
